# KanBoardDS Foundations

As Foundations representam a base visual do KanBoardDS.

Nenhum componente pode definir seus próprios valores de:

- cores
- tipografia
- espaçamento
- radius
- elevação
- animações

Todo componente deve consumir exclusivamente os tokens definidos nesta pasta.

## Estrutura

- colors
- typography
- spacing
- radius
- elevation
- motion
- icons

## Princípios

- Sem números mágicos.
- Sem Color(0xFF...).
- Sem EdgeInsets.all(16).
- Sem BorderRadius.circular(12).
- Sem TextStyle inline.

Tudo deve ser centralizado nas Foundations.
